Image Format Converter

Runs entirely in your browser. Nothing is uploaded anywhere.

Note: Animated GIFs will only convert their first frame — canvas-based conversion can't preserve animation. For animated GIFs, keep the original file if you need the animation.
Note: Source BMP files are supported as input and will convert fine. BMP is not offered as an output format because browsers can't encode it directly — PNG is the closest lossless equivalent.
